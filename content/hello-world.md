## Olá, mundo!

Este é o primeiro post do blog construído com **Next.js 14**, **Styled Components** e **Contentlayer**.

### Por que essa stack?

- **Next.js 14 (App Router):** SSG nativo e Server Components reduzem o bundle JS ao mínimo.
- **Styled Components 6:** CSS-in-JS com escopo por componente e suporte a SSR via registry.
- **Contentlayer:** Transforma arquivos Markdown em dados JSON tipados, garantindo que campos obrigatórios de SEO estejam sempre presentes.
